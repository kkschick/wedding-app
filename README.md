# Wedding App

## Getting Started

# Setup back-end
```
$ git clone git@github.com:kkschick/wedding-app.git
$ cd wedding-app/
$ virtualenv venv
$ . venv/bin/activate
$ pip install -r requirements.txt
$ python server/app.py
 * Running on http://0.0.0.0:5000/
```

# Setup front-end
```
$ npm install
$ npm start
```
