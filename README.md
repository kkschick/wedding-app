# Wedding App

## Getting Started

### Clone repo
```
$ git clone git@github.com:kkschick/wedding-redux.git
$ cd wedding-redux/
```

### Setup back-end
```
$ virtualenv venv
$ . venv/bin/activate
$ pip install -r requirements.txt
$ python server/app.py
 * Running on http://0.0.0.0:5000/
```

### Setup front-end
```
$ npm install
$ npm start
```
